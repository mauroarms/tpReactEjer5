# Ejercicio 5 del Trabajo Práctico de React ⚛ RollingCodeSchool <>

En este programa se creó una lista de tareas que agrega y borra tareas agregadas mediante un form y las va agregando en una lista... Además de que detecta si una tarea está repetida y muestra una advertencia al borrarla... Con diferencia al ejercicio anterior, ahora las tareas son guardadas en el localStorage del navegador
